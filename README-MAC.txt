========================================
HOW TO LAUNCH DOWNDRIVE ON MAC (NATIVE)
========================================
Because YouTube blocks cloud servers, running this natively on your Mac's home Wi-Fi is the absolute best way to extract videos! 

1. Double-click the DownDrive.zip file to extract it.
2. It will create a folder called "DownDrive-main" in your Downloads folder.
3. Open Spotlight (Command ⌘ + Spacebar), type "Terminal", and hit Enter to open your Mac's command line.
4. Copy the entire line of code below, paste it into the Terminal, and hit Enter:

    cd ~/Downloads/DownDrive-main && python3 -m pip install -r requirements.txt && python3 app.py

5. Leave that Terminal window open in the background!
6. Open Safari or Chrome and go to: http://localhost:5000

Enjoy limitless, un-blockable extractions!
